# 🐍 Python 自動化挑戰（7 Days Challenge）

> 一個用 7 天時間，逐步養成寫 Python 自動化腳本習慣的計劃。  
> 每天花 20~30 分鐘，完成一個能在日常或工作中派上用場的小專案。
